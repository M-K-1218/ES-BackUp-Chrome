# ES Backup

Webフォーム上の設問と回答をバックアップ用途でクリップボードへコピーする Chrome 拡張機能です。

## 使い方

1. Chrome の `chrome://extensions` を開く
2. 「デベロッパー モード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」からこのフォルダを選択する
4. フォームページで拡張機能を開き、「設問と回答をコピー」を押す

「Googleドキュメントを開く」を押すと `https://docs.new` を新規タブで開きます。

## 仕様

- 現在表示中のページに存在するフォーム要素のみ取得します。
- 永続保存、履歴管理、クラウド連携は行いません。
- パスワード、hidden、submit、button、reset は取得対象外です。
- 未入力項目は `（未入力）` として出力します。
- 同じ設問名が複数ある場合は 2 件目以降に `(2)` のような連番を付けます。
